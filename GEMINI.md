# Git Tools Extension

Enhances Git workflow with AI-powered commit messages and PR drafts.

## Commands

- `/git-commit:write`: Generates a commit message for staged changes.
- `/git-commit:amend`: Generates a commit message to amend the previous commit.
- `/git-pr:draft`: Generates a PR title and description from branch changes vs target.

## Standards

- **Conventional Commits**: All generated messages must strictly follow the Conventional Commits 1.0.0 specification.
